Plugin created shortcode to insert YouTube videos with microdate on Schema.org for rich snippet

== Description ==

The plugin allows to speed up and simplify the process of partitioning a video from YouTube about writing articles. 
You don't need to know HTML, simply install the plugin and you are in the edit box of the article to receive the button. 
Click on the button and fill in all fields. Click on the "Insert" button and you're done.

https://www.youtube.com/watch?v=SSesEP5-9rM

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the `wp-shortcode` folder to the to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You can see '+' button on post editor, using this button, you can add shortcodes in your posts.
